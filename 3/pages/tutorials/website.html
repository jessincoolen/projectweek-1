Een basis html pagina

In deze tutorial laat ik zien hoe je een basis html website maakt/codeert. Je hebt als eerste een code editor nodig. Ik raad zelf visual studio code aan. Het enige wat gebruikt word is de html code en visual studiocode(dit mag ook een andere code editor zijn).



Previous Next

Je maakt een folder aan die website gaat heten en maakt hier een map aan die je html noemt.
Hierin maak je een file aan die index.html heet. Dit doe je in je code editor.

Voor mensen die visualstudio gebruiken is het vrij makkelijk om te beginnen. Het enigen wat je moet intypen is "html:5"
Voor een andere code editor heb je de code zelf nodig. de code werkt als volg:

<!DOCTYPE html> is er om aan te geven dat je html:5 gebruikt.

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1"> Dit zorgt ervoor dat de website op alle apparaten werkt.

Je kunt de pagina een titel op het tabblad geven door de tag <title>home</title> Nu maak je een body aan. <body> </body> In de body komt alle content te staan:

Je hebt nu dit aan code:

<!DOCTYPE html>
<html>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<body>
</body>
</html>

Nu plaats je in de body de tag <header> hierin zet je de navbar die hieronder staat.
<nav class="w3-bar w3-black">
<a href="#home" class="w3-button w3-bar-item">Home
<a href="#band" class="w3-button w3-bar-item">Band
<a href="#tour" class="w3-button w3-bar-item">Tour
<a href="#contact" class="w3-button w3-bar-item">Contact
</nav>

Nu sluit je de header weer met de volgende tag. </header>

Hierna voeg je dit toe aan de eerste regel van de nav bar. align="center" style="word-spacing: 5px;"
Dit gaat er zo uit zien: <nav align="center" style="word-spacing: 5px;" class="w3-bar w3-black">
De "align='center'" is om de navbar in het midden te krijgen.
De "style="word-spacing: 5px;"" is om de afstand tussen de woorden te bepalen.

Hierna ga je een nieuwe sectie aan maken. Deze sectie word de <main>
Je gaat nu een titel plaatsen. dit doe je met de volgonde code: <h2>Welkom</h2>.
De tekst geef je aan met een <p></p>.
Hier komt de belangrijkste content in. Je kunt hier voor het idee een stukje lorem ipsum in doen.
Je hebt nu dit aan code:

<html>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<title>home</title>
<body>
<header>

Je kunt de breedte van de main ook aanpassen. Dit doe je door in de tag een style toe te voegen.
Dit moet je erin zetten: "style="width: 67%; float: left;"" ook zorg je ervoor dat die links op de pagina staat.

Je kunt ook nog een <aside></aside> in je website zetten.
Deze kun je ook een titel geven. Dit doen we met een <h3>meer informatie</h3>.
Hieronder voeg je weer een <p></p> tag toe.
Hier voegen we weer lorem aan toe.

Als je een afbeelding wilt toevoegen moet je deze tag gebruiken <img src="" alt="">.
Bij src voeg je je de afbeelding toe en bij alt hoef je niks te zetten.

Nu maak je nog een footer aan <footer> aan.
Door een &copy; toe te voegen krijg je een copyright teken.
Je kunt nu je eigen copyright maken met de volgende tag: <p>copyright &copy; 2019 by Jan</p>
Nu kun je de footer sluiten met </footer>.

Je kunt de footer gemakkelijk beneden krijgen door <br> te gebruiken.
Wat de br doet is een regel leeg laten in de tekst.
