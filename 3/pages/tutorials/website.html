een basis html pagina

In deze tutorial laat ik zien hoe je een basis html website maakt/codeert. Je hebt als eerste een code editor nodig. Ik raad zelf visual studio code aan. Het enige wat gebruikt word is de html code en visual studiocode(dit mag ook een andere code editor zijn).




Voor mensen die visualstudio gebruiken is het vrij makkelijk om te beginnen. Het enigen wat je moet intypen is "html:5"
Voor een andere code editor heb je de code zelf nodig. de code werkt als volg:

<!DOCTYPE html> is er om aan te geven dat je html:5 gebruikt.

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1"> Dit zorgt ervoor dat de website op alle apparaten werkt.

Nu maak je een body aan. <body> </body> In de body komt alle content te staan:

Je hebt nu dit aan code.
<!DOCTYPE html>
<html>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<body>
</body>
</html>

Nu plaats je in de body de tag <header> hierin zet je de navbar die hieronder staat.
<nav class="w3-bar w3-black">
<a href="#home" class="w3-button w3-bar-item">Home
<a href="#band" class="w3-button w3-bar-item">Band
<a href="#tour" class="w3-button w3-bar-item">Tour
<a href="#contact" class="w3-button w3-bar-item">Contact
</nav>

Nu sluit je de header weer met de volgende tag. </header>

Hierna voeg je dit toe aan de eerste regel van de nav bar. align="center" style="word-spacing: 5px;"
Dit gaat er zo uit zien: <nav align="center" style="word-spacing: 5px;" class="w3-bar w3-black">
De "align='center'" is om de navbar in het midden te krijgen.
De "style="word-spacing: 5px;"" is om de afstand tussen de woorden te bepalen.
